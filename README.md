## ğŸ“¦ Installation & Setup

### Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/kayspace/morse-warriorz.git
   ```

2. Navigate to the project directory:
   ```bash
   cd morse-warriorz
   ```

3. Open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```
